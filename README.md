# Go Micro Starter

This is a Go Micro starter template

## Overview

Basic starter template gives you best practices for Go Micro services. 

## Features

- [x] Makefile
- [x] Go Mod
- [x] Example call
- [ ] ...

## Usage

Install deps

```
make init
```

### Generating proto

```
make proto
```


### Running service

Build it

```
make build
```

Run it

```
./starter
```
